About Lopes Market

Created by Phillip Ball, Darnae Simmons, and Thomas Spalango, this project is the milestone of our cloud computing class, where our team of three collaborated to develop a website hosted on the cloud. Our goal was to integrate cloud technologies to create a scalable and secure web application. We used cloud-based hosting services, database management, and authentication systems. Throughout this process, we used various cloud computing concepts, including Lambda serverless and API gateways reinforcing our understanding of cloud deployment.

Tech Stack

  • Frontend

    Angular, TypeScript, HTML, CSS

  • Backend

    Node.js, Express, MongoDB, AWS Lambda, API Gateway

  • Authentication

    JWT Authentication

  • Hosting

    Cloudflare Pages
